# ADR: Agentic Communication (acomm)

This directory contains Architectural Decision Records (ADR) for the `acomm` project.

## Decided Records

- [ADR 001: Core Architecture and Tech Stack](./001-core-architecture-and-tech-stack.md) - Rust, Tokio, and Trait-based channel abstraction.
- [ADR 002: Preflight Routing and Metadata Strategy](./002-preflight-routing-and-metadata-strategy.md) - Context-aware routing and security.
- [ADR 003: Terminal Equivalence and Interactive UI](./003-terminal-equivalence-and-interactive-ui.md) - UX goals and real-time feedback.
- [ADR 004: CLI Wrapper Orchestration and TUI Dashboard](./004-cli-wrapper-orchestration-and-tui-dashboard.md) - Minimalist orchestration and TUI implementation.
- [ADR 005: TypeScript/Ink TUI and Unix Socket Bridge Architecture](./005-typescript-ink-tui-and-bridge-architecture.md) - CJK-aware TUI in TypeScript/Ink, Rust binary as Unix socket bridge server.

## Status

- **Proposed**: Decisions under discussion or awaiting initial implementation.
- **Accepted**: Decisions that have been implemented or agreed upon as the foundation.
- **Deprecated/Superseded**: Decisions that are no longer valid.
